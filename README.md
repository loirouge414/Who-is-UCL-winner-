1. 이 프로그램을 만든 이유와 동기
    평소에 해외축구를 보는 것을 좋아하기도 하고, 현재 시즌 중에 있는 유럽 챔피언스리그의 결과를 예측해 보고 싶었다.
   이를 위해, 오픈소스소프트웨어 강의 시간에 배운 점들을 이용하여 API를 가져오고, 이를 기반으로 예측을 할 수 있었기에 이 주제를 선정하였다.
   
2. 구현 과정
    1) 데이터 가져오기 (data_loader.py)
        해외축구의 여러 요소를 종합하여 유럽 클럽의 점수를 자체적으로 매겨 놓은 clubelo.com의 API를 사용하였다.
       API를 호출할 때, 챔피언스리그에 진출하는 팀의 API만 가져오는 것을 원하기에, 챔피언스리그에 진출하는 36개의 팀을 csv 파일에 적고, 이와 매치되는 팀의 API만 호출하게끔 하였다.
       또한, API를 호출하는 clubelo.com 사이트에는 클럽의 이름이 다르게 나와있기에, 클럽의 정식 명칭을 csv 파일에 따로 적었다.
       
    3) 데이터 기반 자체 노이즈 생성 (ratings.py)
        실제 해외축구에는 부상, 불화설, 날씨와 같은 다양한 경기 내외로 영향을 끼치는 요소들이 많다.
       호출된 API를 기반으로 한 클럽의 점수만을 사용하면, 항상 같은 결과가 나오기에, 적절한 노이즈 점수를 난수로 부여하여 0과 1사이의 점수를 각각 부여받게끔 하였다.
  
    4) 파워 점수를 이용하여 각 리그 단계에 맞는 코드 작성
        2)에서 작성한 코드를 바탕으로 먼저 리그페이즈의 결과를 나타냈다.
       1-8위는 16강 진출, 9-24위는 플레이오프 진출, 25-36위는 리그페이즈 탈락과 같은 결과를 나타냈다.
       9-24위는 상위 순위와 하위 순위가 각각 맞붙게끔 하여 승리한 8팀을 선정하였다.
       16강에서는 플레이오프에서 올라온 팀 8팀과 직행하여 올라온 8팀들이 각각 붙어 승리한 8팀을 선정하였다.
       8강에서는 승리한 4팀을 선정하고,
       4강에서도 같이 승리한 2팀을 선정,
       결승에서 승리한 팀을 선정하여 최종 우승팀을 선정하게 하였다.
